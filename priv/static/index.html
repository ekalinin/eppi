Wellcome to the eppi!

eppi - erlang based python package index

This is a PyPI compatible package index.

To use this server with pip, run the the following command: 

$ pip install -i http://127.0.0.1:7890/simple pack1 [pack2...]

To use this server with easy_install, run the the following command: 

$ easy_install -i http://127.0.0.1:7890/simple pack1

The complete list of all packages can be found via the simple index.
